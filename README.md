# Ruby-enumerable

This is a project to recreate ruby enumerable methods :

- `#each`
- `#each_with_index`
- `#select`
- `#all?`
- `#any?`
- `#none?`
- `#count`
- `#map`
- `#inject`

## Built With

- RUBY
- VS Code

## Getting Started

To get a local copy up and running follow these simple steps (you need to have Ruby installed in your machine)

- `git clone <repo link>`
- `cd Ruby-enumerable`
- `ruby my_scripts.rb`
